About this project

     This project was started to gain a practical experience of front end development HTML + CSS+Java script. During this project I want to not only practice with programming languages or CSS but also to use different database systems such as SQLite and MongoDb. In this project I will try to implement modern modern trends and metodologies of front end development, work with google fonts and github. All steps of my front end experiennce I will try to describe in a blog, that will be created on this site.

    I will create a site with a cooking recipes first of all because I love to cook and secondly, because I can try different technologies developing this site. There will be several sections, such as:

  • Blog - the place where will be all my notes according to this project, knowledge, problems and concerns that I am facing. The storage of blog's records I will organize using SQLite.
  • Forum - the section where site users can communicate wich each other. It will be a classic forum and the storage of it's records I will organize using MongoDB.
  • Gallery - the list of photos of different recipes.The place where users could upload their images. In this section I plan to create some animation or effects using Java script. Then this animation could be used as a solid framework.
  • Search - search page with different fiters. The search will be created as for SQLite as for MongoDB.
